# Lifecycle Science/RE2 API




# Introduction
_Improve user engagement with real-time "Next Best Action" recommendations using RE2 from Lifecycle Science._
RE2 enables consumer action recommendations,
message orchestration, and content personalization in a variety of 
marketing and product development use cases. RE2 uses
anonymized event logging to optimize category and asset recommendations for any
user based on their own engagement history.
The RE2 API provides authenticated systems with program owners access to 
the real-time retention and engagement engine built by Lifecycle Science. 

Only two API call integrations are needed to get started:

1. `[POST] /events` for logging user engagement events
1. `[GET] /users/<user_id>/actions` to retrieve user recommended actions.


All of the other API resources available in the RE2 system provide optional configuration and optimization features.
An API Key and Program ID are necessary to access the API. Contact Lifecycle Science support to obtain.
