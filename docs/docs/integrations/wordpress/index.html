You need to set some stuff up.

Install Google Tag Manager​

To install Google Tag Manager on your WordPress website, you will first need to create a Google Tag Manager account and obtain your GTM code. Once you have your code, you can follow these steps to install it on your WordPress site:

  1. Log in to your WordPress website and go to your admin dashboard.
  2. From the left-hand menu, select "Plugins" and then click "Add New".
  3. In the search field, type "Google Tag Manager for WordPress" and hit enter.
  4. From the search results, select the "Google Tag Manager for WordPress" plugin by Thomas Geiger and click "Install Now".
  5. After the plugin has been installed, click "Activate".
  6. Once the plugin is activated, go to the "Settings" section in the left-hand menu and click on "Google Tag Manager".
  7. In the "Google Tag Manager" settings page, enter your GTM code and click "Save Changes".

After following these steps, Google Tag Manager should be installed on your WordPress website. Keep in mind that it may take some time for the changes to take effect.
